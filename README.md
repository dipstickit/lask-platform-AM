# Lask Platform - Angular Admin Panel

### Used technologies
- [Angular](https://angular.io/)
- [Angular Material](https://material.angular.io/)
- [NgRx](https://ngrx.io/)
- [Cypress](https://www.cypress.io/)


## Installation
```bash
npm install
```

## Running the app
```bash
npm run start
```

## Building the app
```bash
npm run build
```
Built app is available in the `dist` folder.

## Test
```bash

npm run test

npm run cypress:open
```
